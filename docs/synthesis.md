# 武器合成システム

## 概要
武器の合成ルールと必要素材数の計算方法を説明します。

## レア度について
- **Legend (L)**: L4, L3, L2, L1
- **Star (S)**: S4, S3, S2, S1
- **Galaxy (G)**: G4, G3, G2, G1
- **Universe (U)**: U4

### レア度の見方
- **X4**: 下級（最も低いランク）
- **X3**: 中級
- **X2**: 上級
- **X1**: 最上級（各ティアで最も高いランク）

## 合成ルール

### Legendティア内の合成
| 素材 | 必要数 | 結果 |
|------|--------|------|
| L4   | 5本    | L3   |
| L3   | 5本    | L2   |
| L2   | 5本    | L1   |

### LegendからStarへの昇格
| 素材 | 必要数 | 結果 |
|------|--------|------|
| L1   | 3本    | S4   |

### Starティア内の合成
| 素材 | 必要数 | 結果 |
|------|--------|------|
| S4   | 3本    | S3   |
| S3   | 3本    | S2   |
| S2   | 2本    | S1   |

### StarからGalaxyへの昇格
| 素材 | 必要数 | 結果 |
|------|--------|------|
| S1   | 2本    | G4   |

### Galaxyティア内の合成
| 素材 | 必要数 | 結果 |
|------|--------|------|
| G4   | 2本    | G3   |
| G3   | 2本    | G2   |
| G2   | 2本    | G1   |

### GalaxyからUniverseへの昇格
| 素材 | 必要数 | 結果 |
|------|--------|------|
| G1   | 2本    | U4   |

## 必要素材数早見表（L1基準）

| 目標武器 | 必要なL1の本数 | 計算式 |
|----------|----------------|--------|
| L4       | (1/125本分)          | 1/ 5³     |
| L3       | (1/25本分)           | 1 / 5²     |
| L2       | (1/5本分)            | 1 / 5     |
| L1       | 1本            | 基準   |
| S4       | 3本            | 3      |
| S3       | 9本            | 3²     |
| S2       | 27本           | 3³     |
| S1       | 54本           | 3³×2   |
| G4       | 108本          | 54×2   |
| G3       | 216本          | 108×2  |
| G2       | 432本          | 216×2  |
| G1       | 864本          | 432×2  |
| U4       | 1,728本        | 864×2  |

## 合成フロー図

```
L4 ×5→ L3 ×5→ L2 ×5→ L1 ×3→ S4 ×3→ S3 ×3→ S2 ×2→ S1 ×2→ G4 ×2→ G3 ×2→ G2 ×2→ G1 ×2→ U4
```

## 合成のポイント

- **L1（ドラゴンブレス バズーカ）**が全ての合成の基準となります
- Legendティアでは5本ずつ合成することで上位レアに進化します
- Starティア以降は2〜3本での合成となり、効率が良くなります
- 最高レアのU4（超小型宇宙原子砲）を作るには、L1が1,728本必要です
